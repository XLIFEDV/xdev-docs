General UI ayarları, arayüzün **temel görünümünü**, **kullanım hissini** ve genel UI davranışlarını kontrol eder.

## Bu bölümde neler yer almalıdır?

Aşağıdaki türde ayarlar bu bölümde belgelenmelidir:

- UI açılış / kapanış animasyonları (yapılandırılabiliyorsa)
- Ses efektlerini açma / kapama seçenekleri (ileride eklenirse)
- UI ölçeklendirme / responsive kuralları (yapılandırılabiliyorsa)
- Genel UI zamanlama değerleri (gecikmeler, geçiş süreleri vb.)

## Önerilen dokümantasyon stili

UI ile ilgili yeni bir ayar eklerken aşağıdaki bilgileri mutlaka ekleyin:

- **Kullanıcının ekranda ne gördüğü**
- **Ayar değiştirildiğinde neyin değiştiği**
- **Ekran görüntüsü / gif** (ileride)
- **Örnek yapılandırma**
