Yerleşim ayarları, POS Cihazı arayüzünün farklı çözünürlüklerde nasıl konumlandığını ve nasıl davrandığını tanımlar.

## Belgelenmesi önerilen başlıklar

- Desteklenen çözünürlükler
- Arayüz ölçeklendirme kuralları
- Güvenli alan / boşluklar (padding)
- Yerleşime özgü ayarlar veya anahtarlar
